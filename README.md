# Eventide Calendar 🌙

A beautifully crafted, modern events calendar built with a focus on aesthetics and user experience. 

![Eventide Preview](https://via.placeholder.com/1200x630.png?text=Eventide+Calendar+Preview)

## ✨ Features

- **Modern UI**: Sleek, accessible, and highly aesthetic interface.
- **Interactive Calendar**: Seamless date selection and event management.
- **Responsive Design**: Looks great on desktop, tablet, and mobile.
- **Form Validation**: Robust event creation with Zod and React Hook Form.

## 🛠️ Tech Stack

- **Framework:** [Next.js 16](https://nextjs.org/) (App Router)
- **Library:** [React 19](https://react.dev/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **UI Components:** [Radix UI](https://www.radix-ui.com/) & [shadcn/ui](https://ui.shadcn.com/)
- **Date Handling:** [date-fns](https://date-fns.org/) & [react-day-picker](https://react-day-picker.js.org/)

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/CodeWithBasu/Eventide-Calender.git
   cd events-calendar-2026
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000) to see the application running.
